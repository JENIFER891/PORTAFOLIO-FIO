Tecnologías de Desarrollo Web

Open Web Platform

Colección de tecnologías abiertas y estándares desarrollados por organismos como W3C, Unicode Consortium, IETF y ECMA International.

Principios:

  • Evitar ser controlado por ninguna empresa
  • Transparencia en todos los niveles
  • Capacidad de integración con fuentes externas
  • Documentación y especificaciones abiertas
  • Libertad de uso
  • Compartir conocimiento (blogs, conferencias)

Estándares Internacionales

Básicos:

  • HTTP/HTTPS: Protocolo para transmisión de documentos hipermedia
  • HTML: Lenguaje para describir y definir contenido web
  • CSS: Hojas de estilo para presentación del contenido
  • XML: Lenguaje de marcado para intercambiar información estructurada
  • JavaScript: Lenguaje de programación para crear sitios interactivos

Complementarios:

  • Web APIs: Para manipular DOM, reproducir multimedia, gráficos 3D
  • SVG: Gráficos vectoriales escalables
  • WebGL: API para gráficos 3D/2D en canvas
  • MathML: Lenguaje para mostrar ecuaciones matemáticas
  • ARIA: Para accesibilidad web
Funcionamiento de la Web

DNS - Sistema de Nombres de Dominio

El DNS es el "directorio telefónico" de Internet que traduce nombres de dominio (como www.uncp.edu.pe) a direcciones IP para que los navegadores puedan cargar los recursos.

Proceso de resolución DNS:

  1. Usuario escribe URL (ej. www.google.com)
  2. Consulta al servidor DNS raíz
  3. Redirección al servidor TLD (.com)
  4. Redirección al servidor de nombres del dominio
  5. Obtención de la dirección IP
  6. Conexión al servidor web
Desarrollo Frontend y Backend

Frontend

Parte de la programación web que se enfoca en la interfaz y experiencia del usuario (HTML, CSS, JavaScript).

Herramientas:

  • Frameworks CSS: Bootstrap, Foundation, Bulma, Materialize
  • Frameworks JS: React, Vue, Angular, Ember
  • Preprocesadores: Sass, Less
  • Build Tools: Webpack, Gulp

Backend

Parte que se encarga de la lógica, base de datos y funcionamiento interno de la aplicación.

Tecnologías:

  • Lenguajes: PHP, Python, Java, Node.js, Ruby
  • Frameworks: Laravel, Django, Spring, Express
  • Bases de datos: MySQL, PostgreSQL, MongoDB
  • Servidores web: Apache, Nginx, Node.js
Diseño UX/UI

Diferencias entre UX y UI

User Experience (UX):

  • Arquitectura de la información
  • Definición de necesidades del usuario
  • Wireframes y prototipos
  • Accesibilidad
  • Enfoque analítico

User Interface (UI):

  • Identidad visual
  • Guía de estilos
  • Color y tipografía
  • Bocetos
  • Enfoque creativo
