Introducción a VSCode

Descarga e Instalación

Visual Studio Code es un editor de código fuente ligero pero poderoso disponible para Windows, macOS y Linux.

# Descargar desde: http://code.visualstudio.com

Primeros Pasos

  • Nuevo archivo: Archivo → Nuevo Archivo
  • Abrir archivo: Archivo → Abrir Archivo
  • Abrir carpeta: Archivo → Abrir Carpeta
  • Archivos recientes: Mostrados en la pantalla de inicio
Interfaz de VSCode

Pantalla Principal

  • Barra de menús: Archivo, Editar, Selección, Ver, etc.
  • Barra lateral: Explorador de archivos, búsqueda, control de versiones, etc.
  • Barra de estado: Información sobre el archivo actual, rama Git, etc.
  • Panel: Terminal, problemas, salida, consola de depuración

Áreas Principales

  • Explorador: Navegación por archivos y carpetas
  • Editor: Área principal para editar código
  • Terminal integrada: Para ejecutar comandos
  • Paleta de comandos: Acceso rápido a funciones (Ctrl+Shift+P)
Funcionalidades Clave

Paleta de Comandos (Ctrl+Shift+P)

Permite acceder rápidamente a todas las funciones de VSCode:

  • Crear nueva terminal
  • Añadir comentarios de línea
  • Eliminar etiquetas con Emmet
  • Configurar extensiones

Emmet

Plugin para generación rápida de código HTML y CSS mediante abreviaciones:

Operador Significado Ejemplo
> Hijo (dentro de...) div>p → <div><p></p></div>
+ Hermano (al lado de...) div+p → <div></div><p></p>
^ Salir de la etiqueta actual div>p^span → <div><p></p></div><span></span>
* Multiplicador ul>li*3 → <ul><li></li><li></li><li></li></ul>
. Clase div.container → <div class="container"></div>
# ID div#main → <div id="main"></div>
Atajos de Teclado

Generales

Comando Acción
Ctrl+Shift+P, F1 Mostrar paleta de comandos
Ctrl+P Abrir rápido archivo
Ctrl+, Abrir configuración de usuario
Ctrl+K Ctrl+S Atajos de teclado

Edición Básica

Comando Acción
Ctrl+X Cortar línea
Ctrl+C Copiar línea
Alt+↑/↓ Mover línea arriba/abajo
Ctrl+Shift+K Eliminar línea
Ctrl+Enter Insertar línea abajo
Ctrl+Shift+Enter Insertar línea arriba

Búsqueda y Reemplazo

Comando Acción
Ctrl+F Buscar
Ctrl+H Reemplazar
F3/Shift+F3 Siguiente/anterior coincidencia
Alt+Enter Seleccionar todas las coincidencias
