Variables CSS y Custom Properties

Definición y uso

Las propiedades personalizadas (variables CSS) permiten definir valores reutilizables en todo el documento.

:root { --main-color: #4a3fb8; --padding-default: 15px; } .elemento { color: var(--main-color); padding: var(--padding-default); }

Ventajas

  • Facilitan el mantenimiento del código
  • Permiten cambios globales con una sola modificación
  • Mejoran la consistencia del diseño
  • Pueden ser modificadas con JavaScript
Funciones CSS

calc()

Permite realizar cálculos matemáticos directamente en CSS.

.elemento { width: calc(100% - 80px); }

clamp()

Define un valor entre un mínimo y un máximo, con un valor ideal.

.elemento { font-size: clamp(1rem, 2.5vw, 2rem); }

min() y max()

Seleccionan el valor mínimo o máximo entre varios parámetros.

.elemento { width: min(50vw, 200px); height: max(40em, 98%); }
Grid Layout Avanzado

Áreas de Grid

Permite definir áreas con nombres para organizar el layout.

.contenedor { display: grid; grid-template-areas: "header header header" "sidebar main main" "footer footer footer"; } .header { grid-area: header; } .sidebar { grid-area: sidebar; } .main { grid-area: main; } .footer { grid-area: footer; }

auto-fit vs auto-fill

auto-fit auto-fill
Ajusta columnas al espacio disponible Crea tantas columnas como quepan
Columnas vacías colapsan Mantiene columnas vacías
Ideal para contenido dinámico Útil cuando se espera más contenido
Flexbox Avanzado

Alineación

  • justify-content: Alineación en eje principal
  • align-items: Alineación en eje transversal
  • align-content: Alineación de múltiples líneas
  • align-self: Alineación individual de items

Distribución Dinámica

.item { flex-grow: 1; /* Capacidad de crecimiento */ flex-shrink: 1; /* Capacidad de reducción */ flex-basis: 100px; /* Tamaño inicial */ }
Diseño Responsivo

Media Queries

Permiten aplicar estilos condicionales según características del dispositivo.

/* Mobile-first */ body { font-size: 16px; } @media (min-width: 768px) { body { font-size: 18px; } } @media (min-width: 1200px) { body { font-size: 20px; } }

Enfoque Mobile-First

  • Diseñar primero para móviles
  • Progresivamente mejorar para pantallas más grandes
  • Prioriza simplicidad y contenido esencial
  • Mejora la velocidad de carga
Scroll Snap y Técnicas de Scroll

Scroll Snap

Permite "anclar" elementos durante el desplazamiento.

.container { scroll-snap-type: y mandatory; overflow-y: scroll; } .item { scroll-snap-align: start; }

Scroll Suave

html { scroll-behavior: smooth; }
