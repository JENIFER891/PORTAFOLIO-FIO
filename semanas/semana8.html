MONOGRAFÍA JAVASCRIPT

Introducción a JavaScript

¿Qué es JavaScript?

Lenguaje de scripting que permite implementar funcionalidades dinámicas en sitios web. Se ejecuta principalmente del lado del cliente.

Ejecución en Navegadores

// El navegador carga el HTML // Encuentra etiquetas <script> y ejecuta el código // El motor JS interpreta y ejecuta el código

Motores de JavaScript

  • Chrome/Node.js: V8
  • Firefox: SpiderMonkey
  • Safari: JavaScriptCore
Variables y Tipos de Datos

Declaración de Variables

let variable = "valor"; // Alcance de bloque const constante = 10; // No se puede reasignar var antiguo = true; // No recomendado (alcance de función)

Tipos de Datos Primitivos

// String, Number, Boolean, Null, Undefined, Symbol, BigInt let nombre = "Juan"; let edad = 30; let esEstudiante = true; let id = Symbol("id");

Tipos No Primitivos

// Object, Array, Function let persona = {nombre: "Ana", edad: 25}; let frutas = ["manzana", "plátano"]; function saludar() { return "Hola"; }
Operadores

Aritméticos

let a = 10, b = 3; console.log(a + b); // 13 (suma) console.log(a % b); // 1 (módulo) a++; // Incremento

Lógicos

// && (AND), || (OR), ! (NOT) if (edad >= 18 && tieneLicencia) { console.log("Puede conducir"); }

Comparación

// == (igualdad débil), === (igualdad estricta) // >, <, >=, <= if (edad === "30") // false (tipos diferentes)
Estructuras de Control

Condicionales

// if/else if (hora < 12) { console.log("Buenos días"); } else { console.log("Buenas tardes"); } // switch switch(dia) { case 1: console.log("Lunes"); break; default: console.log("Día no válido"); }

Bucles

// for for (let i = 0; i < 5; i++) { console.log(i); } // while while (condicion) { // código } // do-while do { // código ejecutado al menos una vez } while (condicion);
Funciones

Tipos de Funciones

// Declaración function suma(a, b) { return a + b; } // Expresión const resta = function(a, b) { return a - b; }; // Flecha const multiplicar = (a, b) => a * b;

Parámetros y Retorno

// Parámetros predeterminados function saludar(nombre = "Invitado") { return `Hola, ${nombre}`; } // Retorno de objeto function crearPersona() { return { nombre: "Ana", edad: 25 }; }
Manipulación del DOM

Selección de Elementos

document.getElementById('miId'); document.querySelector('.miClase'); document.querySelectorAll('div');

Modificación del DOM

// Cambiar contenido elemento.textContent = "Nuevo texto"; elemento.innerHTML = "<strong>Texto</strong>"; // Manipular clases elemento.classList.add('nueva-clase'); elemento.classList.remove('clase-vieja'); // Crear elementos const nuevoDiv = document.createElement('div'); document.body.appendChild(nuevoDiv);
Eventos y Almacenamiento

Manejadores de Eventos

// Click boton.addEventListener('click', function() { console.log("Botón clickeado"); }); // Formulario formulario.addEventListener('submit', function(e) { e.preventDefault(); // Evitar recarga // Procesar formulario });

Almacenamiento Local

// localStorage (persistente) localStorage.setItem('usuario', JSON.stringify({nombre: "Ana"})); const usuario = JSON.parse(localStorage.getItem('usuario')); // sessionStorage (solo sesión) sessionStorage.setItem('config', JSON.stringify({tema: "oscuro"}));
