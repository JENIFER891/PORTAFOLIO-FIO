Semana 5: Introducción a React

Biblioteca de JavaScript para construir interfaces de usuario interactivas y reutilizables.

Qué es React

Definición y características

  • Biblioteca de JavaScript para crear interfaces de usuario
  • Se utiliza para construir aplicaciones de una sola página (SPA)
  • Permite crear componentes reutilizables
  • Desarrollado por Facebook y mantenido por la comunidad

Documentación oficial

Documentación General | Inicio Rápido

Historia de React

Linea de tiempo

  • 2011: Primera implementación en el Newsfeed de Facebook
  • Creado por Jordan Walke, ingeniero de Facebook
  • Julio 2013: Lanzamiento público (v0.3.0)
  • Abril 2022: Versión actual v18.0.0

Versiones importantes

Versión Lanzamiento Características principales
v0.3.0 Julio 2013 Primera versión pública
v16.0.0 Sept 2017 Fiber architecture
v18.0.0 Abril 2022 Concurrent rendering
Desarrollo con React usando CDN

Incluir React desde CDN

<!-- Versión 17 --> <script src="https://unpkg.com/react@17/umd/react.development.js" crossorigin></script> <script src="https://unpkg.com/react-dom@17/umd/react-dom.development.js" crossorigin></script> <script src="https://unpkg.com/@babel/standalone/babel.min.js"></script> <!-- Versión 18 --> <script crossorigin src="https://unpkg.com/react@18/umd/react.development.js"></script> <script crossorigin src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>

Ejemplo básico

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Ejemplo React</title> <!-- Incluir React --> </head> <body> <div id="root"></div> <script type="text/babel"> function App() { return <h1>Hola Mundo React!</h1>; } ReactDOM.render(<App />, document.getElementById('root')); </script> </body> </html>

Creación de un proyecto React

Create React App

# Crear proyecto npx create-react-app nombreProyecto # Entrar al directorio cd nombreProyecto # Iniciar servidor de desarrollo npm start

Vite

# Crear proyecto npm init vite@latest # Seleccionar opciones # Ingresar nombre del proyecto # Seleccionar React # Instalar dependencias cd nombreProyecto npm install # Iniciar servidor npm run dev

Next.js

# Crear proyecto npx create-next-app # Ingresar nombre del proyecto cd nombreProyecto # Iniciar servidor npm run dev
Estructura de un proyecto React

Create React App

  • node_modules/ - Dependencias del proyecto
  • public/ - Archivos estáticos
  • src/ - Código fuente principal
  • package.json - Configuración del proyecto

Vite

  • src/ - Contiene los componentes principales
  • vite.config.js - Configuración de Vite
  • index.html - Punto de entrada

Next.js

  • pages/ - Rutas de la aplicación
  • public/ - Archivos estáticos
  • styles/ - Archivos CSS globales
