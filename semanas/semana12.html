Semana 12: Desarrollo Backend con PHP y Laravel

Exploramos el desarrollo backend con PHP, Apache y el framework Laravel.

Configuración del Entorno

Apache HTTP Server

Apache es el servidor web más popular para entornos PHP. Configuración básica:

# httpd.conf Listen 8080 ServerName localhost:8080 Define SRVROOT "D:\server\Apache24" DocumentRoot "${SRVROOT}/htdocs"
Archivo de configuración httpd.conf

Instalación de PHP

Configuración del archivo php.ini:

extension_dir = "ext" extension=mysqli extension=curl extension=fileinfo extension=pdo_mysql

Verificar instalación con: php -v

Composer - Gestor de dependencias

Instalación y verificación:

# Instalar Composer php -r "copy('https://getcomposer.org/installer', 'composer-setup.php');" php composer-setup.php # Verificar composer -V

Laravel Framework

Instalación de Laravel

# Crear nuevo proyecto composer create-project laravel/laravel mi-proyecto # Iniciar servidor cd mi-proyecto php artisan serve
Proceso de instalación de Laravel

Estructura de Directorios

  • app/: Contiene la lógica de la aplicación (controladores, modelos)
  • routes/: Definición de rutas (web.php, api.php)
  • resources/: Vistas y assets (CSS, JS)
  • database/: Migraciones, factories y seeders
  • config/: Archivos de configuración
Estructura de directorios en Laravel

Conceptos Clave

Eloquent ORM: Sistema de mapeo objeto-relacional para trabajar con bases de datos

Blade: Motor de plantillas para crear vistas

Artisan: CLI para automatizar tareas

Middleware: Filtros para peticiones HTTP

Ejemplo Práctico: CRUD Estudiantes

1. Crear proyecto y configurar base de datos

laravel new estudiantes-app cd estudiantes-app # Configurar .env DB_DATABASE=academico DB_USERNAME=root DB_PASSWORD=tu_contraseña

2. Crear migración para modelo Estudiante

php artisan make:model Estudiante -m # Editar migración Schema::create('estudiantes', function (Blueprint $table) { $table->id('idEstudiante'); $table->string('nomEstudiante'); $table->string('dirEstudiante'); $table->string('ciuEstudiante'); $table->timestamps(); }); # Ejecutar migración php artisan migrate

3. Configurar Tailwind CSS

npm install -D tailwindcss@3 postcss autoprefixer npx tailwindcss init -p # Configurar tailwind.config.js content: [ "./resources/**/*.blade.php", "./resources/**/*.js", "./resources/**/*.vue", ], # Añadir a resources/css/app.css @tailwind base; @tailwind components; @tailwind utilities; # Compilar npm install npm run dev

4. Crear rutas, controlador y vista

# routes/web.php use App\Http\Controllers\EstudianteController; Route::get('/', [EstudianteController::class, 'create']); Route::post('/guardar', [EstudianteController::class, 'store']); # Controlador php artisan make:controller EstudianteController public function create() { return view('formulario'); } public function store(Request $request) { Estudiante::create([ 'nomEstudiante' => $request->nombre, 'dirEstudiante' => $request->direccion, 'ciuEstudiante' => $request->ciudad ]); return redirect('/')->with('mensaje', 'Estudiante registrado'); }
