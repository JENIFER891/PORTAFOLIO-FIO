Reflexión

Este portafolio ha sido una valiosa oportunidad para organizar, integrar y reflexionar sobre todo lo aprendido a lo largo del curso. A medida que fui desarrollando cada sección semanal, no solo reforcé mis conocimientos técnicos en herramientas como HTML, CSS y React, sino que también comprendí la importancia de documentar mi aprendizaje de forma clara y estructurada. Gracias a este proceso, mejoré notablemente en la creación de interfaces, en el uso de GitHub Pages para publicar mis proyectos, y en el dominio de conceptos fundamentales como el uso de componentes, hooks, y la gestión del estado en React. Cada semana representó un reto distinto, desde el pensamiento computacional y el diseño visual, hasta la creación dinámica de aplicaciones con JavaScript moderno. Además, este portafolio me permitió reflexionar sobre mi crecimiento personal y profesional: reconocer lo que antes me costaba, apreciar lo que ahora domino con mayor fluidez, y plantearme nuevas metas a partir de lo aprendido. Sin duda, ha sido una herramienta útil no solo para mostrar mi trabajo, sino también para fortalecer mi compromiso con el aprendizaje continuo.
